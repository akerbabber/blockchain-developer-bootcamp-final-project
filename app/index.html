Yet Another OTC Exchange

Please connect to the Kovan testnet to use this dApp

Yet Another OTC Exchange is an OTC (Over the counter) decentralised (is on the blockchain and there is no admin keys) exchange, developed for the ConsenSys Blockchain Developer Bootcamp final project by Maurizio Murru. It makes the user load their tokens on a set that can be later exchanged for another set.
